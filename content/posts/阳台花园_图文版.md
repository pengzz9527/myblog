## 第二章：蔬菜种植实战

### 图片4：生菜生长过程图
**图片描述：**
- 时间轴展示：从种子到收获的完整过程
- 第1周：发芽期（小嫩芽）
- 第2-3周：生长期（展开真叶）
- 第4-6周：成熟期（可采摘）
- 第7周后：持续收获（边摘边长）

**种植步骤：**
```
📅 时间安排：春秋季最佳
🌱 播种：浅播，覆薄土，保持湿润
🌿 育苗：3-4片真叶时移栽
💧 浇水：土壤表面干即浇
🌞 光照：半日照，避免暴晒
🍃 收获：外叶先摘，促进新叶生长
```

### 图片5：番茄种植示意图
**图片描述：**
- 番茄植株结构图：标注根系、主干、侧枝、花序
- 支撑系统：竹竿支撑、网格爬藤、吊篮悬挂
- 果实特写：不同成熟阶段的番茄

**关键技巧：**
1. **掐尖打顶：** 保留3-4个果穗后掐顶
2. **疏花疏果：** 每穗保留4-5个果实
3. **追肥时机：** 开花前、坐果后、膨大期
4. **病虫害防治：** 提前预防，生物防治

🌿 薄荷：繁殖快，需单独盆栽

## 第四章：养护与管理

### 图片10：浇水技巧图解
**图片描述：**
- 正确浇水：根部浇透，叶面干燥
- 错误示范：叶面浇水导致病害
- 浇水工具：长嘴壶、滴灌系统、喷雾瓶
- 浇水时机：早晚凉爽时

**浇水原则：**
```
💦 见干见湿：表面干再浇水
💦 浇则浇透：直到底部出水
💦 因植制宜：喜湿多浇，耐旱少浇
💦 季节调整：夏季勤，冬季少
```

### 图片11：病虫害识别图
**图片描述：**
- 常见病害：白粉病、黑斑病、根腐病
- 常见虫害：蚜虫、红蜘蛛、白粉虱
- 防治方法：物理防治、生物防治、药剂防治
- 预防措施：合理密植、通风透光

**有机防治方法：**
```
🐞 天敌引入：瓢虫吃蚜虫
🌿 植物驱虫：薄荷驱蚂蚁，大蒜防病害
🍶 自制药剂：辣椒水、大蒜水、肥皂水
🌬️ 物理方法：手工捕捉、黄板诱杀
```

### 图片12：修剪整形示范
**图片描述：**
- 修剪前后对比：株型更美观，通风更好
- 修剪工具：修枝剪、园艺锯、手套
- 修剪部位：枯枝、病枝、交叉枝、徒长枝
- 修剪时机：生长期轻剪，休眠期重剪

**修剪原则：**
```
✂️ 保持自然：不过度修剪
✂️ 促进生长：剪弱留强
✂️ 改善通风：疏除密集枝条
✂️ 控制株型：定期整形
```
